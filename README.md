# 🌧️ AlertaRisk – Plataforma de Prevenção a Alagamentos
🚨 Monitoramento de zonas de risco e alertas para prevenir catástrofes causadas pelas chuvas.

## 📌 Introdução
O AlertaRisk é uma plataforma inovadora voltada para a prevenção e mitigação de desastres naturais, utilizando dados meteorológicos e geográficos para prever e alertar sobre alagamentos em áreas de risco.

Nosso propósito é salvar vidas e minimizar danos, garantindo que a população esteja preparada antes, durante e depois de uma emergência.

## ⚙️ Funcionalidades
🖥️ Página Inicial
✔️ Cadastro de usuários em áreas de risco
✔️ Notificações instantâneas via WhatsApp e SMS
✔️ Medidas preventivas e orientações de segurança
✔️ Acesso rápido a contatos de emergência: SAMU, Bombeiros, Defesa Civil e Polícia Militar de Pernambuco

### 📌 Módulos do Sistema
✅ Cadastro e Login → Criação de conta, autenticação segura e recuperação de senha
✅ Mapa Interativo → Exibição de zonas de risco em tempo real e alertas ativos
✅ Alertas Inteligentes → Monitoramento contínuo e notificações automáticas sobre chuvas intensas e alagamentos
✅ Fórum Comunitário → Um espaço dinâmico e colaborativo para a troca de informações entre usuários. As postagens são organizadas por categorias, como:
- Alagamentos
- Deslizamentos de Terra
- Bueiros Abertos
- Outros Desastres Naturais Relacionados à Chuva
✅ Orientações de Segurança → Diretrizes claras sobre como agir antes, durante e depois de um evento extremo

## 🛠️ Tecnologias Utilizadas
### 🎨 Front-end:

- HTML e CSS

- React.js

### 🚀 Back-end:

- Java
- Node.js
